1. 3D Solar System.

This project is an interactive 3D representation of the solar system, designed to educate users about the planets and their orbits. Utilizing WebGL and JavaScript, the application allows users to explore each planet in detail, showcasing features such as size comparisons, orbital paths, and unique characteristics. Users can navigate through space, zoom in on planets, and access informative pop-ups that provide facts about each celestial body. This project highlights my skills in 3D modeling and interactive design, demonstrating my ability to create engaging educational tools.
